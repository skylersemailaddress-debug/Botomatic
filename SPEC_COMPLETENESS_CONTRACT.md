# Spec Completeness Contract

Build is blocked unless all are true:
- critical completeness >= 95
- commercial completeness >= 90
- unresolved high-risk questions = 0
- Build Contract exists
- high-risk assumptions are approved or explicitly accepted
- no critical contradiction exists

High-risk areas always require explicit confirmation:
- payments
- legal/compliance
- auth/security
- permissions and access
- data deletion and retention
- regulated workflows
- external API cost liability
- public/private visibility
- irreversible business rules
