# MVP Scope

## Goal

First working loop:
chat -> master truth -> plan -> execute packet -> CI -> preview -> status

## Included
- chat intake via GPT
- master truth compilation
- architecture approval
- milestone + packet generation
- single executor path
- GitHub integration
- GitHub Actions validation
- Vercel preview status
- project status reporting

## Excluded (retained for later)
- full autonomy across all packets
- production auto-release
- multi-repo orchestration
- advanced proof bundles
- broad stack support

## Success Criteria
- user can describe app in chat
- system produces structured architecture
- system executes at least one packet automatically
- CI passes or returns failure status
- preview URL is produced
- status is visible in chat
